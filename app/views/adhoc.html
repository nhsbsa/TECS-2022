Unable to set up Direct Debit

The current version loops the user back to the 'pay by Direct Debit' screen on clicking 'Continue', this version proposes instead to redirect the user to be able to select which payment method they would like to use with the hypothesis being if they are redirected to the start of pay by Direct Debit journey again they might feel like they can't pay when in reality they could pay by card as an alternative option.

What is your address? (Postcode look up)

There are various screens throughout the journeys which ask a user to fill out their address, the hypothesis is by implementing an address look up via postcode, we could reduce time to complete by eliminating the need to type out an address, and also reduce input errors. We would also like to maintain the option to add an address manually for hard to find / obscure addresses.
